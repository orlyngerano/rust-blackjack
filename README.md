# Blackjack

Terminal based game.


## Usage


##### Starts the game.
```
% ./blackjack
```

##### Show more options when running the game.
```
% ./blackjack --help
```

## Installing Build Tools

Follow Rust installation [Guide](https://www.rust-lang.org/tools/install)

## Building

```
cargo build --release
```
The built file is located in directory **target/release**.


## Testing

```
cargo test
```
